# e3650_pac_tool

A collection of tools for packaging and preparing E3650-related PAC files.

## Directory Overview

- `bootloader/`: Directory for e3650 bootloader.
- `flashloader/`: Directory for flashloader files.
- `builtin_tools/`: Directory for built-in tools (such as `pactool`, `bin2hex`, and `atb_signer`).
- `keys/`: Directory for signing or encryption-related keys.

## Important Note

Files in the `bootloader/` directory (for example, `simple_bootloader.pac`) are built from the following repository:

https://github.com/leon6002/e3650_bootloader
